# 🐾 Lịch tập 4 tuần · Kate

Trang web lịch tập 4 tuần (tăng cơ · giảm mỡ · siết nét), mỗi buổi 45–60 phút.
Mỗi bài tập là **link tìm kiếm YouTube** — luôn hoạt động, không bao giờ chết link.

## Nội dung
- `index.html` — trang web trực quan (4 tab tuần, bấm tập trực tiếp)
- `workout-schedule-kate.ics` — file lịch import vào Google / Apple Calendar
- `workout-schedule-kate.md` — bản Markdown dạng bảng

## Xem online
Sau khi bật GitHub Pages: `https://<username>.github.io/<repo>/`

## Bật GitHub Pages
1. Push repo này lên GitHub (public).
2. Vào **Settings → Pages**.
3. Mục **Build and deployment → Source** chọn **Deploy from a branch**.
4. Branch chọn **main** / thư mục **/ (root)** → **Save**.
5. Đợi ~1 phút, link web hiện ở đầu trang Pages.
